## 🚀 How to Run

From inside an exercise folder, run a file with:

```bash
python3 <filename>.py
```

Example:

```bash
python3 ex00.py
```

If a specific exercise requires arguments:

```bash
python3 <filename>.py arg1 arg2
```

---

## 🧪 Testing

Most exercises can be tested directly from the terminal.

Typical workflow:

```bash
python3 <filename>.py
python3 <filename>.py "example input"
```

When relevant, edge cases to test include:

* empty input
* invalid arguments
* unexpected types
* boundary values
